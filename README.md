## Getting Started

### Installing dependencies

```bash
bun install
```

### Running the app

First, run the development server:

```bash
bun run dev
```

### Building for production

```bash
bun run build
```

### Running tests

```bash
bun run test
```

See the full [testing guide](./playwright-tests/README.md).
